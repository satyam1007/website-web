About This Project

This portfolio features three interactive web applications designed to simplify everyday tasks. The Attendance Tracker helps record and manage student attendance efficiently, the Expense Tracker allows users to track daily spending and manage budgets, and the Task Manager makes it easier to organize and complete daily to-dos.

Attendance Tracker — Smart Attendance Web App

A modern attendance web appbuilt using HTML, CSS, and JavaScript for quick, responsive, and easy attendance tracking.
